# CGL 智能助手 - 安装指南

如果是 Chrome 或 Edge 浏览器，请按照以下步骤安装：

## 第一步：获取插件包
请向管理员索取 `cgl-extension-v1.0.0.zip` 压缩包，并将其**解压**到一个文件夹中（例如放在桌面的 `CGL工具` 文件夹）。

## 第二步：在浏览器中加载

### 🔵 Google Chrome (谷歌浏览器)
1. 在地址栏输入 `chrome://extensions` 并回车。
2. 打开右上角的 **"开发者模式" (Developer mode)** 开关。
3. 点击左上角的 **"加载已解压的扩展程序" (Load unpacked)** 按钮。
4. 选择您刚才**解压后**的文件夹（注意选文件夹，不是选文件）。
5. ✅ 安装完成！

### 🟢 Microsoft Edge
1. 在地址栏输入 `edge://extensions` 并回车。
2. 打开左侧菜单底部的 **"开发人员模式"** 开关。
3. 点击 **"加载解压缩的扩展"** 按钮。
4. 选择您刚才**解压后**的文件夹。
5. ✅ 安装完成！

---
**常见问题：**
- **Q: 为什么图标是灰色的？**
  A: 只有在亚马逊 (Amazon) 的相关页面上，插件才会激活变亮。
- **Q: 更新了新版本怎么办？**
  A: 删除旧文件夹，下载新包解压，然后在扩展页面点击“重新加载”图标即可。
