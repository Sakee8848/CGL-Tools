# Cybreturn CGL 智能助手 - 安装指南

## 📌 插件简介

**Cybreturn CGL 智能助手** 是一款专为亚马逊卖家设计的浏览器插件，提供以下功能：

✅ **智能报价分析**：上传销售报告（支持 Excel、CSV、TXT、PDF、Word），自动计算 CGL 保费  
✅ **一键抓取数据**：在亚马逊卖家中心页面直接提取 SKU 和销售数据  
✅ **快捷访问**：一键打开完整版 Web 工具进行深度分析  

---

## 🚀 安装步骤（开发者模式）

### 1. 解压文件
将收到的 `CGL_Extension.zip` 解压到您的电脑任意位置（建议放在不会误删的文件夹）。

### 2. 打开 Chrome 扩展程序管理页面
在 Chrome 浏览器地址栏输入：
```
chrome://extensions/
```
或者点击右上角 **⋮** → **扩展程序** → **管理扩展程序**

### 3. 开启开发者模式
在页面右上角找到 **"开发者模式"** 开关，将其打开（变成蓝色）。

### 4. 加载插件
点击左上角的 **"加载已解压的扩展程序"** 按钮。

### 5. 选择文件夹
在弹出的文件选择窗口中，找到并选择刚才解压的 **`extension`** 文件夹（不是 ZIP 文件本身）。

### 6. 完成安装
插件会立即出现在扩展程序列表中，浏览器工具栏右上角会显示插件图标 🛡️。

## 🎯 使用方法

### 📊 上传销售报告
1. 点击浏览器工具栏的插件图标
2. 点击 **"选择文件 (支持所有格式)"**
3. 上传您的销售报告（Excel、CSV、TXT、PDF、Word 均可）
4. 查看预估保费结果

### 🔍 抓取亚马逊页面数据
1. 打开亚马逊卖家中心的业务报告页面
2. 点击插件图标
3. 点击 **"从亚马逊页面提取数据"**
4. 系统会自动识别并提取页面中的 SKU 和销售数据

### 🔗 打开完整版工具
点击 **"打开完整版工具"** 按钮，跳转到完整的 Web 应用进行深度分析。

## ⚠️ 常见问题

### Q1: 为什么会提示"请停用以开发者模式运行的扩展程序"？
**A:** 这是 Chrome 对开发者模式插件的正常提示。点击"取消"即可继续使用，不影响功能。如需去除提示，请使用 Chrome Web Store 官方版本。

### Q2: 插件图标不显示怎么办？
**A:** 点击浏览器工具栏右侧的 **拼图图标**，找到 "Cybreturn CGL 智能助手"，点击 **📌 图钉** 将其固定到工具栏。

### Q3: 抓取功能提示"未检测到数据"？
**A:** 请确保您在亚马逊卖家中心（Seller Central）的业务报告页面。普通购物页面无法提取销售数据。

### Q4: 支持哪些文件格式？
**A:** 
- ✅ Excel (.xlsx, .xls)
- ✅ CSV (.csv)
- ✅ 文本文件 (.txt)
- ✅ PDF (.pdf)
- ✅ Word (.docx, .doc)

### Q5: 如何更新插件？
**A:** 
1. 打开 `chrome://extensions/`
2. 找到 "Cybreturn CGL 智能助手"
3. 点击右下角的 **刷新图标** ⟳
4. 或者重新加载新版本的文件夹
